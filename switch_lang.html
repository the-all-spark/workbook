Задача: Как переключиться на другой язык страницы?

Task: How to switch to a different language on the page?

Предположим, у нас имеется мультиязычная страница. Тогда при клике на соответствующую кнопку (RU или EN) пользователь может изменить язык отображаемого текста.
Вот этот переключатель. Попробуйте!

Let's assume that we have a multi-language page. In this case, by clicking on the corresponding button (RU or EN) the user can change the language of the displayed text.
Here is this switcher. Try it!

Переключите язык | Switch the language:

По умолчанию, для отображения страницы установлен русский язык (кнопка переключения RU заблокирована), но можно предоставить пользователю возможность самому выбрать, на каком языке ему изначально показать страницу, например, через форму. Поиграться с такой формой можно по ссылке (проект в Codepen)

By default, Russian language was set to display the page (the RU switch button is blocked), but you can give the user the opportunity to choose in which language the page should be initially displayed, for example, via a form.
You can check such a form at the link (project in Codepen).

Итак, как сделать переключение между языками страницы?

So, how to switch between the languages of the page?

<span class="switch">Переключите язык:</span>
<button class="ru-btn" disabled>RU</button>
<button class="en-btn">EN</button>

<div class="ru-text">
    Здесь блок, включающий русский текст. 
</div> 
<div class="en-text">
    This is block with English text. 
</div>  
<div class="ru-text">
    Здесь еще блок русского текста. 
</div> 
<div class="en-text">
    And another English text.
</div>
/* Отображение контента */
.ru-text {
    display: block;
}
.en-text {
    display: none;
}
// Переключение языка
let engTexts = document.querySelectorAll(".en-text");
let rusTexts = document.querySelectorAll(".ru-text");
let engButton = document.querySelector(".en-btn");
let rusButton = document.querySelector(".ru-btn");
 
// при клике на кнопку EN
engButton.addEventListener("click", switchToEnglish);
// при клике на кнопку RU
rusButton.addEventListener("click", switchToRussian);

function switchToEnglish() {
   for(let i= 0; i < rusTexts.length; i++) {
      rusTexts[i].style.display = "none";
   }
   for(let j= 0; j < engTexts.length; j++) {
      engTexts[j].style.display = "block";
   }
   engButton.setAttribute("disabled", "");
   rusButton.removeAttribute("disabled");
}

function switchToRussian() {      
   for(let i= 0; i < rusTexts.length; i++) {
      rusTexts[i].style.display = "block";
   }
   for(let j= 0; j < engTexts.length; j++) {
      engTexts[j].style.display = "none";
   }
   rusButton.setAttribute("disabled", "");
   engButton.removeAttribute("disabled"); 
}

Hover on a list item to highlight the corresponding code.

    В HTML-коде:

  1. Cоздать 2 кнопки - RU и EN - с классами <button class="ru-btn" disabled> и <button class="en-btn">. Одну из кнопок сделать неактивной (disabled) (по умолчанию это кнопка переключения на русский язык - RU);
  2. Тексту на русском языке (всем элементам <h1>, <div>, <p>, <span>, <ol>, <ul> и т.д.) присвоить class="ru-text" (например, <div class="ru-text">).
  3. Тексту на английском языке присвоить class="en-text" (например, <div class="en-text">).

    4. В файле CSS:

  4. Элементы с классом class="ru-text" изначально отобразить (display: block;), а с class="en-text" - скрыть (display: none;).

    5. В файле JS:

  5. Получить доступ к коллекции элементов на русском и английском языке, а также к кнопкам RU и EN через document, querySelectorAll и querySelector. Сохранить в переменных.
  6. Кнопкам добавить обработчик событий (addEventListener) - по клику ("click") запускать функцию switchToEnglish или switchToRussian.
  7. В функции switchToEnglish перебрать все элементы с русским текстом и скрыть их (rusTexts[i].style.display = "none";), а элементы с английским текстом, наоборот, отобразить (engTexts[j].style.display = "block";).
    В функции switchToRussian наоборот.
  8. В функции switchToEnglish кнопку EN заблокировать (через setAttribute) - "disabled", а с кнопки RU убрать атрибут "disabled", чтобы она снова стала активной (через removeAttribute).
    В функции switchToRussian наоборот.

В результате:

  • При клике на кнопку EN будет отображаться контент на английском языке. Сама кнопка станет недоступной.
  • При клике на кнопку RU - контент на русском языке. При этом кнопка станет недоступной.
