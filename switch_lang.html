Задача: Как переключиться на другой язык страницы?

Task: How to switch to a different language on the page?

Предположим, у нас имеется мультиязычная страница. Тогда при клике на соответствующую кнопку (RU или EN) пользователь может изменить язык отображаемого текста.
Вот этот переключатель. Попробуйте!

Let's assume that we have a multi-language page. In this case, by clicking on the corresponding button (RU or EN) the user can change the language of the displayed text.
Here is this switcher. Try it!

Переключите язык | Switch the language:

По умолчанию, для отображения страницы установлен русский язык (кнопка переключения RU заблокирована), но можно предоставить пользователю возможность самому выбрать, на каком языке ему изначально показать страницу, например, через форму. Поиграться с такой формой можно по ссылке (проект в Codepen)

Итак, как сделать переключение между языками страницы?

So, how to switch between the languages of the page?

<span>Переключите язык:</span>
<button class="ru-btn" disabled>RU</button>
<button class="en-btn">EN</button>

<div class="ru-text">
    Здесь блок, включающий русский текст. 
</div> 
<div class="en-text">
    This is block with English text. 
</div>  
<div class="ru-text">
    Здесь еще блок русского текста. 
</div> 
<div class="en-text">
    And another English text.
</div>
/* Отображение контента */
.ru-text {
    display: block;
}
.en-text {
    display: none;
}
// Переключение языка
let engTexts = document.querySelectorAll(".en-text");
let rusTexts = document.querySelectorAll(".ru-text");
let engButton = document.querySelector(".en-btn");
let rusButton = document.querySelector(".ru-btn");
 
// при клике на кнопку EN
engButton.addEventListener("click", switchToEnglish);
// при клике на кнопку RU
rusButton.addEventListener("click", switchToRussian);

function switchToEnglish() {
   for(let i= 0; i < rusTexts.length; i++) {
      rusTexts[i].style.display = "none";
   }
   for(let j= 0; j < engTexts.length; j++) {
      engTexts[j].style.display = "block";
   }
   engButton.setAttribute("disabled", "");
   rusButton.removeAttribute("disabled");
}

function switchToRussian() {      
   for(let i= 0; i < rusTexts.length; i++) {
      rusTexts[i].style.display = "block";
   }
   for(let j= 0; j < engTexts.length; j++) {
      engTexts[j].style.display = "none";
   }
   rusButton.setAttribute("disabled", "");
   engButton.removeAttribute("disabled"); 
}

Hover on a list item to highlight the corresponding code.

  1. обернуть всю страницу в блок, например, <div class="container">;
  2. сделать из блока-обертки flex-контейнер (display: flex;);
  3. в блок-обертку вложить <header>, <main> и <footer>;
  4. родителю (блоку-обертке) задать flex-direction: column; чтобы блоки встали друг под другом, а не в ряд;
  5. родителю задать min-height: 100vh;, чтобы он занимал как минимум всю высоту экрана;
  6. блоку main задать flex: 1; (или flex-grow: 1;)

В результате:

  • Центральный блок страницы (<main>) будет всегда растягиваться на всё доступное свободное пространство.
  • Из-за этого <footer> всегда будет прижиматься к нижнему краю страницы.
  • При этом, если контента в <main> будет больше чем на один экран, то он растянется и подстроится под весь объем.
